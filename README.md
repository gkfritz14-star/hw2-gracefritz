# hw2-gracefritz

## Project Overview

This project automates the generation of sales follow-up emails from raw post-meeting notes.

**Workflow:**
- **Input:** Unstructured notes from a sales call
- **Output:** A polished, professional follow-up email that thanks the prospect for their time and highlights key next steps or touchpoints in the sales pipeline

**Value:**
- Saves time for salespeople managing high volumes of post-call follow-ups
- Ensures consistent tone and quality across the sales team
